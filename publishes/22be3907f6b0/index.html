Java 虚拟机（JVM）配置选项说明（JDK8+）

标准选项（Standard Options）

这些是Java虚拟机（JVM）的所有实现都支持的最常用的选项。

选项和默认值 描述 示例
-agentlib:libname[=options] 加载本地的代理类 -agentlib:jdwp=transport=dt_socket,server=y,address=8000
-agentpath:pathname[=options] 加载绝对路径的本地的代理类
和-agentlib类似，区别是使用绝对路径
-client 使用Java HotSpot客户端虚拟机
-Dproperty=value 设置系统属性的值
注意：属性名不可以有空格，属性值可以有空格（有空格的时候用英文双引号 " 括起来，如："foo bar"）
-disableassertions[:[packagename]…|:classname]
-da[:[packagename]…|:classname]		 禁用断言
默认情况下所有包中所有类里面的断言都是被禁用的		 -da
-da:com.wombat.fruitbat.Brickbat MyClass
-disablesystemassertions
-dsa		 禁用所有系统类中的断言
-enableassertions[:[packagename]…|:classname]
-ea[:[packagename]…|:classname]		 启用断言
默认情况下所有包中所有类里面的断言都是被禁用的
-enablesystemassertions
-esa		 启用所有系统类中的断言
-help
-?		 在不运行JVM的情况下，显示java命令的使用信息
-jar filename 执行封装成JAR文件的程序 java -jar foo.jar
-javaagent:jarpath[=options] 加载指定的Java语言代理
-jre-restrict-search 在版本搜索中包括用户私有的JRE
-no-jre-restrict-search 在版本搜索中排除用户私有的JRE
-server 使用Java HotSpot服务器虚拟机
-showversion 显示版本信息，并且继续执行程序
-splash:imgname 通过指定的图片来显示启动画面 -splash:images/splash.gif
-verbose:class 显示每个加载的类的信息
-verbose:gc 显示每次垃圾回收（GC）事件的信息
-verbose:jni 显示本地方法（native methods）的使用信息和其它本地接口（JNI, Java Native Interface）的活动信息
-version 显示Java版本信息，然后退出程序
-version:release 指定运行程序使用的发行版本 -version:”1.6.0_13 1.6* & 1.6.0_10+”

非标准选项（Non-Standard Options）

这些选项是Java热点虚拟机（Java HotSpot VM）特定的通用选项。

选项和默认值 描述 示例
-X 显示所有-X选项的帮助信息
-Xbatch 禁用后台编译
默认情况下，JVM编译方法是通过后台任务完成的。此选项会使JVM使用前台任务的方式来编译方法，等同于-XX:-BackgroundCompilation

-Xbootclasspath/a:path
-Xbootclasspath/p:path
-Xcheck:jni
-Xcomp
-Xdebug
-Xdiag
-Xfuture
-Xint
-Xinternalversion
-Xloggc:filename
-Xmaxjitcodesize=size
-Xmixed
-Xmnsize
-Xmssize
-Xmxsize
-Xnoclassgc
-Xprof
-Xrs
-Xshare:mode
-XshowSettings:category
-Xsssize
-Xverify:mode
